# 🏏 IPL Data Analysis (2008-2025)

Exploratory Data Analysis on ball-by-ball IPL data using Python, Pandas, and Seaborn.

## 📊 Analysis Done
- Top 10 Run Scorers of all time
- Top 10 Wicket Takers of all time
- Season-wise Total Runs trend (2008-2025)
- Team-wise Total Runs comparison

## 🛠️ Tools & Libraries Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📁 Dataset
Ball-by-ball IPL data (2008-2025) from Kaggle — 64 columns, 900,000+ rows

## 🔍 Key Insights
- V Kohli is the all-time leading run scorer in IPL history
- YS Chahal leads the wicket-takers chart with 220+ wickets
- IPL runs jumped significantly after 2022 when 2 new teams were added
- Mumbai Indians have scored the most total runs across all seasons

## 📌 How to Run
1. Download IPL.csv from Kaggle
2. Place it in the same folder as the notebook
3. Run `ipl_analysis.ipynb` cell by cell
